# Bibliography

The DH-lab bibliography lives in Zotero. 

To generate static bibliography for this page, use the `export bibliography` option in the Zotero app, and save it to the root folder of this project. Make sure to use the [custom bibliography style](bibliography/custom_bibliography_style.csl)[^1].

Convert the html to markdown using the [markdowninfy Python script](bibliography/markdownify_bibliography.py). This also adds page metadata, and turns the first line into a header.

[^1]: Made with [Visual editor](https://editor.citationstyles.org/visualEditor/)
